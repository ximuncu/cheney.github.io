ES6中的Promise对象

次访问 条评论
2017-04-07 javascript

Promise是异步编程的一种解决方案。ES6中提供了一个Promise的构造函数用于创建Promise实例。

Promise对象状态

Promise对象状态有3个

1、Pending（进行中）
2、Resolved（已完成，又称 Fulfilled)
3、Rejected（已失败）

状态的改变只有两种情况

1、从Pending变为Resolved
2、从Pending变为Rejected

状态到了Resolved或者Rejected会一直保持。

创建Promise对象

Promise是一个构造函数，使用很简单

Promise构造函数接受一个函数作为参数，该函数的两个参数分别是resolve和reject。它们是两个函数，由JavaScript引擎提供，不用自己部署。

resolve 是指在Promise对象状态 从Pending变为Resolved 时，将参数传出

reject 是指在Promise对象状态 从Pending变为Rejected 时，将参数传出

Promise.prototype.then()

Promise实例有个then方法，分别指定Resolved状态和Rejected状态的回调函数。

下面是一个Ajax操作实例

then方法返回一个全新的Promise实例，可以采用链式写法。

Promise.prototype.catch()

Promise.prototype.catch方法是.then(null, rejection)的别名

Promise.all

Promise.all方法用于将多个Promise实例，包装成一个新的Promise实例。

这个可以认为是一个也不能少，也就是：

1、所有的Promise实例 状态转换为Resolved 则新的Promise实例状态才会转换为 Resolved
2、只有其中有一个Promise实例 状态转换为Rejected，则新的Promise实例状态才会转换为Rejected

Promise.race

Promise.race方法同样是将多个Promise实例，包装成一个新的Promise实例。

和名字一样，谁先改变新的Promise实例也跟着改变

Promise.resolve

Promise.resolve可以将现有对象转为Promise对象

Promise.reject

Promise.reject(reason)方法也会返回一个新的 Promise 实例，该实例的状态为rejected。

finally(),done()

done方法放在链式写法最后，会捕捉到任何可能出现的错误，并向全局抛出。

finally方法用于，不管Promise对象最后状态如何，都会执行的操作
