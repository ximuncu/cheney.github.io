jQuery中的deferred对象

次访问 条评论
2017-04-09 javascript

ES6中有个promise对象，而在这个之前，jQuery中有个deferred对象来实现差不多的功能。deferred是延时的意思，也就是当我们遇到耗时较长的操作，比方说异步读取数据时，我们延时到某个时候再执行。1.5版本以后的jQuery中ajax返回的都是deferred对象。可以进行链式操作

我们常用的一些关于deferred对象的方法

1、jQuery.Deferred() 生成一个deferred对象
2、deferred.done() 状态修改为resolved时回调函数
3、deferred.fail() 状态修改为rejected时回调函数
4、deferred.then() 将2、3两个回调合在一起写
5、deferred.promise() 没有参数返回没有改变状态接口的新的deferred对象，有参数时，给该对象部署deferred接口
6、deferred.resolve() 手动改变状态为 resolved，立即触发done()方法。5中返回的deferred对象没有这个方法
7、deferred.reject() 手动改变状态为 rejected，立即触发fail()方法。5中返回的deferred对象没有这个方法
8、$.when() 将多个deferred对象 包装，如何都resolved 则触发 done；如果存在 rejected 则触发 fail
9、deferred.always() 只要状态改变了就执行
